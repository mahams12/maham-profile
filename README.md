# Maham Saeed - Portfolio

A beautiful, animated portfolio website showcasing my work as a Software Engineer.

## Features

- **Modern Design**: Dark theme with warm coral accents
- **Smooth Animations**: Scroll-triggered reveal effects and hover interactions
- **Responsive**: Works beautifully on desktop, tablet, and mobile
- **Sections**: About, Experience, Projects, Education, Certifications, Strengths, Contact

## Quick Start

1. Open `index.html` in your browser
2. Or use a local server: `python -m http.server 8000` or `npx serve .`

## Structure

```
├── index.html      # Main HTML structure
├── styles.css      # Styling and animations
├── script.js       # Interactivity and scroll effects
├── assets/
│   └── profile.png # Profile photo
└── README.md
```

## Customization

- **Colors**: Edit CSS variables in `:root` in `styles.css`
- **Content**: Update text directly in `index.html`
- **Profile Photo**: Replace `assets/profile.png` with your image
